Help & User Guide

🔬 Overview

This application is a computational chemistry tool designed to extract chemical data, analyze properties, and predict skin absorption using scientific models.

🚀 Main Features

📷 Chemical Scanner (OCR + Camera)

Upload an image or use camera to scan chemical names. The system extracts text automatically.

🧪 Chemical Data Fetch (PubChem)

Retrieve Molecular Weight (MW) and logP using online database.

💧 Skin Absorption Calculator

Predict permeability (Kp) using Potts & Guy equation.

📊 How to Use

1. Upload image OR open camera
2. Extract chemical name
3. Fetch chemical data
4. Click Calculate
5. View results

🧠 Scientific Background

The system uses QSAR-based modeling. Skin permeability depends on:

  • logP (lipophilicity)
  • Molecular Weight (MW)

Higher logP → better membrane penetration
Higher MW → lower penetration

⚠️ Important Notes

Ensure correct chemical name. OCR may require manual correction.
Internet connection required for PubChem data.
